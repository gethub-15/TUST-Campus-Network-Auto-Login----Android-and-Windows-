# 天津科技大学校园网一键登录

一个面向 Android 手机和 Android TV/电视盒子的天津科技大学校园网登录小工具。应用提供校园网登录、注销、在线状态检测、USS 流量查询和流量刷新等功能。

## 功能

- 打开应用时自动检测当前账号是否已连接。
- 支持校园网、中国联通、中国移动、中国电信上网方式选择。
- 支持手动联网、注销、刷新流量。
- 显示总流量、剩余流量、已用流量和在线设备数。
- 适配 Android TV 遥控器焦点操作。

## 隐私说明

本仓库不包含任何个人账号、密码、HAR 抓包文件、签名密钥。首次使用时请在应用界面自行填写学号和密码，软件会记录上次填写内容，但软件不会上传你的任何个人账户信息。

校园网登录接口使用学校网关提供的 HTTP GET 参数方式，账号密码会按网关要求提交给校园网认证服务器。
