# 天津科技大学校园网一键登录

一个用于天津科技大学校园网认证的轻量化登录工具，支持 **Android 手机、Android TV / 电视盒子** 和 **Windows 桌面端**。

本工具主要用于简化校园网网页登录流程，提供校园网登录、注销、在线状态检测、USS 流量查询和流量刷新等功能。

---

## 项目简介

天津科技大学校园网一键登录工具可以帮助用户在常用设备上快速完成校园网认证，避免每次手动打开网页登录页面输入账号密码。

目前项目包含两个版本：

- **Android 版**
  - 适用于 Android 手机、Android TV、电视盒子等设备
  - 支持遥控器焦点操作，适合大屏设备使用

- **Windows 版**
  - 适用于 Windows 电脑
  - 提供独立 exe 程序，可直接运行

---

## 功能特性

- 支持校园网登录
- 支持校园网注销
- 支持在线状态检测
- 支持 USS 流量查询
- 支持手动刷新流量
- 支持显示：
  - 总流量
  - 剩余流量
  - 已用流量
  - 当前在线设备数

## 隐私说明

本仓库不包含任何个人账号、密码、HAR 抓包文件、Cookie、Token 或签名密钥。

首次使用时，请在应用界面自行填写学号和密码。

### Android 版

Android 版会记录上次填写的登录信息，方便下次使用。软件不会将你的个人账户信息上传到除校园网认证服务器以外的任何第三方服务器。

### Windows 版

Windows 版在勾选“保存账号密码到本机”后，会将配置保存在当前 Windows 用户目录下：

```text
%APPDATA%\TustCampusLogin\config.json
```
